# チャネル

```
ch:=make(chan int)
```

同じ型のチャネルは==で比較可能

send,receiveという操作が可能

```
ch <- x
x =<-ch
<-ch //結果は破棄
```

チャネルはクローズで明示的にこれ以上送信されないことを設定できる。

送信を試みるとパニック

# バッファなしチャネル

```
ch=make(chan int)
ch=make(chan int,0)
```

送信、受信しているゴルーチンは同期される。
